Common Windows run commands


Just a quick ref of misc commands.

taskmgr.exe (Task Manager)
taskschd.msc (Task Scheduler)
gpedit.msc (Group Policy Editor)
secpol.msc (Local Security Policy)
sysdm.cpl (System properties, env path, RDP, CN etc)
netplwiz.exe (User Accounts)
eventvwr.msc (Event Viewer)
iscsicpl.exe (ISCSI-Initiator Properties, SAN targets)
intl.cpl (System Language Settings)
devmgmt.msc (Device Manager)
compmgmt.msc (Computer Management)
control panel; control system; etc.
ncpa.cpl (Network Connections)
inetcpl.cpl (Internet Properties)
